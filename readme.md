Задание на воспроизведение эксперимента Мак-Гурка и Мак-Дональда

## Начало работы

Для начала, установите [PyschoPy v.2025.2.3](https://www.psychopy.org/download.html)

- Откройте `\mcgurk.psyexp` в среде PsychoPy Builder для работы с GUI, или же
- Откройте `\mcgurk.py` в среде PsychoPy Coder для работы с кодом

## Дополнительная информация

Файл `\conditions.xlsx` содержит логику предъявления стимулов и всю необходимую meta-информацию

Папка `/data/` будет создана автоматически после завершения эксперимента для сохранения данных
