Задание на воспроизведение эксперимента Мак-Гурка и Мак-Дональда

## Начало работы

Для начала, установите [PyschoPy v.2025.2.3](https://www.psychopy.org/download.html)

- [Скачайте](https://github.com/talasius/mcgurk-experiment/releases/tag/1.0b) архив с экспериментом со страницы последнего релиза
- Распакуйте ахрхив в любую папку
- Откройте `\mcgurk.psyexp` в среде PsychoPy Builder для работы с GUI, или же
- Откройте `\mcgurk.py` в среде PsychoPy Coder для работы с кодом

## Дополнительная информация

Файл `\conditions.xlsx` содержит логику предъявления стимулов и всю необходимую meta-информацию, не меняйте, если не уверены в последствиях

Папка `/data/` будет создана автоматически после завершения эксперимента для сохранения данных

## Важно!

❌ Не запускайте онлайн версию эксперимента (JavaScript) - она не поддерживается

✅ Для корректного функционирования запускайте только `\mcgurk.psyexp` в среде PsychoPy Builder или `\mcgurk.py` в среде PsychoPy Coder
